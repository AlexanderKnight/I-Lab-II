import numpy as np
import matplotlib.pyplot as plt
from scipy.optimize import curve_fit

%matplotlib inline

from matplotlib import rc

#defines needed functions

def gaussianFit (x, a, c, theta):
    return a*np.e**(-(x-c)**2/(2*theta**2))

def linearFit(t, m, b):
    return m*t + b

def quadraticFit(t, a, b, c):
    return a*(t**2)+(b*t)+c

def cosFit(t, a, omega, c):
    return a*np.cos(omega*(t-c))

When we plot u as a function of x in linear, semi-log, and log-log plots, we can see it is linear in the log-log plot. This means the that $u \propto x^\alpha$, with $\alpha$ being a constant. The steeper the slope on the log-log plot, the larger the value of $\alpha$.
